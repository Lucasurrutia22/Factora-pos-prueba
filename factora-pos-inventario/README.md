# Factora POS - Inventario

## Descripción
Este proyecto es un sistema de gestión de inventario llamado "Factora POS". Permite a los usuarios administrar productos, stock y números de serie de manera eficiente. La interfaz incluye un panel lateral, un encabezado, una barra de acciones y una tabla de productos, así como un modal para agregar o editar productos.

## Estructura del Proyecto
El proyecto contiene los siguientes archivos:

- **Inventario.html**: Estructura HTML para la interfaz de gestión de inventario.
- **Inventario.css**: Estilos para la interfaz de gestión de inventario.
- **Inventario.js**: Lógica JavaScript para la gestión del inventario.
- **package.json**: Archivo de configuración para npm.
- **.gitignore**: Especifica los archivos y directorios que deben ser ignorados por Git.
- **README.md**: Documentación del proyecto.

## Instalación
1. Clona el repositorio en tu máquina local.
2. Navega al directorio del proyecto.
3. Instala las dependencias necesarias ejecutando:
   ```
   npm install
   ```

## Uso
1. Abre el archivo `Inventario.html` en un navegador web.
2. Utiliza la interfaz para agregar, editar y gestionar productos en el inventario.
